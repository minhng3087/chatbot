# react-redmine

### Prerequisites

- [Git](https://git-scm.com/)
- [NodeJS 12 or newer](https://nodejs.org/)
- [Yarn 1.5 or newer](https://yarnpkg.com/lang/en/docs/install/)

### Installation

```sh
git clone https://github.com/minhng3087/chatbot
cd chatbot
```

#### Using npm:
```sh
npm install
```

#### Using yarn:

```sh
yarn install
```

### Usage

```sh
expo start
```
